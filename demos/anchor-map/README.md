# anchor-map

First live demo for `show`.

`anchor-map` is a static visual map of the 5 Class A anchors:
- run
- bench
- lab
- show
- pulse

## Why this exists

This is the first visible proof surface inside `show`.
It gives a stranger an immediate picture of the anchor layer and routes them directly into the live repos.

## Open

Open `index.html` in a browser.

## What it shows

- one card per anchor
- role statement
- direct GitHub route
- why the anchor matters in the field

## Notes

This is intentionally simple.
Its job is not to be the final site.
Its job is to make the anchor layer visible now.
